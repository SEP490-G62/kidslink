# UC-SA-005: Quản lý khiếu nại/phản hồi / Manage Complaints and Feedback

## Tiếng Việt
- **Diễn giải:** School Admin tiếp nhận, xử lý, phản hồi khiếu nại từ phụ huynh.
- **Diễn viên chính:** School Admin
- **Diễn viên phụ:** Phụ huynh, Hệ thống
- **Mô tả:** Quản lý khiếu nại, phản hồi từ phụ huynh về các vấn đề trong trường.
- **Điều kiện tiên quyết:**
  - School Admin đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Khiếu nại/phản hồi được tiếp nhận, xử lý và phản hồi lại cho phụ huynh.
- **Luồng chính:**
  1. School Admin truy cập chức năng quản lý khiếu nại.
  2. Xem danh sách khiếu nại/phản hồi.
  3. Xử lý và gửi phản hồi cho phụ huynh.
- **Luồng thay thế:**
  - Khiếu nại không hợp lệ:
    1. Hệ thống báo lỗi.
    2. School Admin xác nhận lại với phụ huynh.

## English
- **Description:** School Admin receives, processes, and responds to complaints from parents.
- **Primary Actor:** School Admin
- **Secondary Actors:** Parent, System
- **Goal:** Manage complaints and feedback from parents about school issues.
- **Preconditions:**
  - School Admin is logged in.
- **Postconditions:**
  - Complaint/feedback is received, processed, and responded to the parent.
- **Normal Flow:**
  1. School Admin accesses the complaint management function.
  2. Views the list of complaints/feedback.
  3. Processes and sends a response to the parent.
- **Alternative Flows:**
  - Invalid complaint:
    1. System notifies error.
    2. School Admin confirms again with the parent.

# UC-SA-004: Quản lý học sinh / Manage Students

## Tiếng Việt
- **Diễn giải:** School Admin quản lý danh sách học sinh, thêm mới, chỉnh sửa, chuyển lớp, xóa học sinh.
- **Diễn viên chính:** School Admin
- **Diễn viên phụ:** Giáo viên, Hệ thống
- **Mô tả:** Quản lý thông tin cá nhân, lớp, trạng thái, chuyển lớp, xóa học sinh khỏi hệ thống.
- **Điều kiện tiên quyết:**
  - School Admin đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Danh sách học sinh được cập nhật chính xác.
- **Luồng chính:**
  1. School Admin truy cập chức năng quản lý học sinh.
  2. Xem danh sách học sinh theo lớp.
  3. Thêm mới, chỉnh sửa, chuyển lớp hoặc xóa học sinh.
  4. Hệ thống cập nhật danh sách học sinh.
- **Luồng thay thế:**
  - Thông tin không hợp lệ hoặc thao tác thất bại:
    1. Hệ thống báo lỗi và yêu cầu nhập lại.

## English
- **Description:** School Admin manages student list, adds, edits, transfers, or deletes students.
- **Primary Actor:** School Admin
- **Secondary Actors:** Teacher, System
- **Goal:** Manage student information, class, status, transfer, and remove students from the system.
- **Preconditions:**
  - School Admin is logged in.
- **Postconditions:**
  - Student list is updated accurately.
- **Normal Flow:**
  1. School Admin accesses the student management function.
  2. Views student list by class.
  3. Adds, edits, transfers, or deletes students.
  4. System updates the student list.
- **Alternative Flows:**
  - Invalid information or operation failed:
    1. System notifies error and requests input again.

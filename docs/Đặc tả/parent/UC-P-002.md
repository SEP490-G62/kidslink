# UC-P-002: Xem báo cáo ngày của con

## Tiếng Việt
- **Diễn giải:** Phụ huynh xem báo cáo ngày về hoạt động, nhận xét của giáo viên đối với con.
- **Diễn viên chính:** Phụ huynh
- **Diễn viên phụ:** Giáo viên, Hệ thống
- **Mô tả:** Xem chi tiết báo cáo ngày của con
- **Điều kiện tiên quyết:**
  - Phụ huynh đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Báo cáo ngày của con được hiển thị đầy đủ, chính xác.
- **Luồng chính:**
  1. Phụ huynh truy cập mục "Báo cáo hàng ngày".
  2. Hệ thống hiển thị danh sách các ngày có báo cáo.
  3. Phụ huynh chọn ngày muốn xem.
  4. Hệ thống hiển thị chi tiết báo cáo ngày của con.
- **Luồng thay thế:**
  - Không có báo cáo:
    1. Hệ thống thông báo không có dữ liệu.

## English
- **Description:** Parent views daily report about child's activities, health, and teacher's comments.
- **Primary Actor:** Parent
- **Secondary Actors:** Teacher, System
- **Goal:** View detailed daily report: meals, sleep, health, activities, comments.
- **Preconditions:**
  - Parent is logged in.
- **Postconditions:**
  - Child's daily report is displayed completely and accurately.
- **Normal Flow:**
  1. Parent accesses the "Daily Report" section.
  2. System shows the list of days with reports.
  3. Parent selects the day to view.
  4. System displays the detailed daily report for the child.
- **Alternative Flows:**
  - No report found:
    1. System notifies that there is no data.

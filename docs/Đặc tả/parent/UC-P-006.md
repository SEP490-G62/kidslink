# UC-P-006: Xem, tạo, bình luận, thích bài đăng (bảng tin)

## Tiếng Việt
- **Diễn giải:** Phụ huynh xem, tạo bài đăng, bình luận, thích bài đăng trên bảng tin lớp/trường.
- **Diễn viên chính:** Phụ huynh
- **Diễn viên phụ:** Giáo viên, Hệ thống
- **Mô tả:** Tương tác với bảng tin: xem, tạo bài đăng, bình luận, thích, xem hình ảnh.
- **Điều kiện tiên quyết:**
  - Phụ huynh đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Bài đăng, bình luận, lượt thích được cập nhật.
- **Luồng chính:**
  1. Phụ huynh truy cập mục "Trang chủ" (bảng tin).
  2. Xem danh sách bài đăng, hình ảnh.
  3. Tạo bài đăng mới, bình luận, thích bài đăng.
  4. Hệ thống cập nhật và hiển thị lại bảng tin.
- **Luồng thay thế:**
  - Không có dữ liệu hoặc gửi thất bại:
    1. Hệ thống báo lỗi và yêu cầu thao tác lại.

## English
- **Description:** Parent views, creates, comments, likes posts on class/school newsfeed.
- **Primary Actor:** Parent
- **Secondary Actors:** Teacher, System
- **Goal:** Interact with newsfeed: view, create posts, comment, like, view images.
- **Preconditions:**
  - Parent is logged in.
- **Postconditions:**
  - Posts, comments, likes are updated.
- **Normal Flow:**
  1. Parent accesses the "Home" (newsfeed) section.
  2. Views list of posts, images.
  3. Creates new post, comments, likes posts.
  4. System updates and displays the newsfeed.
- **Alternative Flows:**
  - No data or send failed:
    1. System notifies error and requests retry.

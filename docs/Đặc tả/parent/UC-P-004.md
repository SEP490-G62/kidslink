# UC-P-004: Xem lịch học/lịch hoạt động của lớp

## Tiếng Việt
- **Diễn giải:** Phụ huynh xem thời khóa biểu, lịch học, lịch hoạt động của lớp con.
- **Diễn viên chính:** Phụ huynh
- **Diễn viên phụ:** Giáo viên, Hệ thống
- **Mô tả:** Xem chi tiết lịch học, hoạt động, sự kiện của lớp.
- **Điều kiện tiên quyết:**
  - Phụ huynh đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Lịch học/lịch hoạt động được hiển thị đầy đủ, chính xác.
- **Luồng chính:**
  1. Phụ huynh truy cập mục "Lịch lớp học".
  2. Hệ thống hiển thị chi tiết lịch học, hoạt động, sự kiện của lớp.
- **Luồng thay thế:**
  - Không có dữ liệu:
    1. Hệ thống thông báo không có dữ liệu.

## English
- **Description:** Parent views class schedule/activity calendar.
- **Primary Actor:** Parent
- **Secondary Actors:** Teacher, System
- **Goal:** View detailed class schedule, activities, and events.
- **Preconditions:**
  - Parent is logged in.
- **Postconditions:**
  - Class schedule/activity calendar is displayed completely and accurately.
- **Normal Flow:**
  1. Parent accesses the "Class Schedule" section.
  2. System displays detailed class schedule, activities, and events.
- **Alternative Flows:**
  - No data found:
    1. System notifies that there is no data.

# UC-P-003: Xem thực đơn của lớp/con

## Tiếng Việt
- **Diễn giải:** Phụ huynh xem thực đơn ăn uống của lớp hoặc của con.
- **Diễn viên chính:** Phụ huynh
- **Diễn viên phụ:** Giáo viên, Hệ thống
- **Mô tả:** Xem chi tiết thực đơn theo ngày/tuần của lớp hoặc của con.
- **Điều kiện tiên quyết:**
  - Phụ huynh đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Thực đơn được hiển thị đầy đủ, chính xác.
- **Luồng chính:**
  1. Phụ huynh truy cập mục "Thực đơn".
  2. Hệ thống hiển thị thực đơn theo ngày/tuần của lớp/con.
- **Luồng thay thế:**
  - Không có thực đơn:
    1. Hệ thống thông báo không có dữ liệu.

## English
- **Description:** Parent views class/child's meal menu.
- **Primary Actor:** Parent
- **Secondary Actors:** Teacher, System
- **Goal:** View detailed daily/weekly menu for class or child.
- **Preconditions:**
  - Parent is logged in.
- **Postconditions:**
  - Menu is displayed completely and accurately.
- **Normal Flow:**
  1. Parent accesses the "Menu" section.
  2. System displays the daily/weekly menu for the class/child.
- **Alternative Flows:**
  - No menu found:
    1. System notifies that there is no data.

# UC-T-006: Chat với phụ huynh

## Tiếng Việt
- **Diễn giải:** Giáo viên có thể gửi và nhận tin nhắn với phụ huynh học sinh trong lớp.
- **Diễn viên chính:** Giáo viên
- **Diễn viên phụ:** Hệ thống, Phụ huynh
- **Mô tả:**
  - Xem danh sách phụ huynh học sinh trong lớp.
  - Gửi, nhận tin nhắn với từng phụ huynh.
  - Xem lịch sử trò chuyện.
- **Điều kiện tiên quyết:**
  - Giáo viên đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Tin nhắn được gửi/nhận thành công, lịch sử trò chuyện được lưu trữ.
- **Luồng chính:**
  1. Giáo viên truy cập mục "Nhắn tin".
  2. Hệ thống hiển thị danh sách phụ huynh học sinh.
  3. Giáo viên chọn phụ huynh để trò chuyện.
  4. Giáo viên gửi/nhận tin nhắn với phụ huynh.
  5. Hệ thống cập nhật lịch sử trò chuyện.
- **Luồng thay thế:**
  - Lỗi gửi/nhận tin nhắn: Hệ thống báo lỗi và giữ nguyên trạng thái.

## English
- **Description:** Teacher can send and receive messages with parents of students in the class.
- **Primary Actor:** Teacher
- **Secondary Actors:** System, Parent
- **Details:**
  - View list of parents of students in the class.
  - Send and receive messages with each parent.
  - View chat history.
- **Preconditions:**
  - Teacher is logged in.
- **Postconditions:**
  - Messages are sent/received successfully, chat history is stored.
- **Normal Flow:**
  1. Teacher accesses the "Messaging" section.
  2. System displays the list of parents.
  3. Teacher selects a parent to chat with.
  4. Teacher sends/receives messages with the parent.
  5. System updates the chat history.
- **Alternative Flows:**
  - Error sending/receiving messages: System shows error and keeps the current state.

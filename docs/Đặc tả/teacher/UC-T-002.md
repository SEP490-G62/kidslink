# UC-T-002: Quản lý lớp học, xem danh sách học sinh

## Tiếng Việt
- **Diễn giải:** Giáo viên có thể xem thông tin lớp học mình phụ trách và danh sách học sinh trong lớp.
- **Diễn viên chính:** Giáo viên
- **Diễn viên phụ:** Hệ thống
- **Mô tả:**
  - Xem thông tin tổng quan về lớp học (tên lớp, sĩ số, giáo viên chủ nhiệm, ...).
  - Xem danh sách học sinh trong lớp (họ tên, mã số, trạng thái, ...).
  - Xem chi tiết hồ sơ từng học sinh.
- **Điều kiện tiên quyết:**
  - Giáo viên đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Thông tin lớp học và danh sách học sinh được hiển thị đầy đủ, chính xác.
- **Luồng chính:**
  1. Giáo viên truy cập mục "Lớp học".
  2. Hệ thống hiển thị thông tin tổng quan về lớp học.
  3. Giáo viên xem danh sách học sinh trong lớp.
  4. Giáo viên chọn học sinh để xem chi tiết hồ sơ (nếu cần).
- **Luồng thay thế:**
  - Không có học sinh nào: Hệ thống hiển thị thông báo "Không có học sinh trong lớp".
  - Lỗi tải dữ liệu: Hệ thống báo lỗi và giữ nguyên trạng thái.

## English
- **Description:** Teacher can view information about their class and the list of students in the class.
- **Primary Actor:** Teacher
- **Secondary Actor:** System
- **Details:**
  - View class overview (class name, number of students, homeroom teacher, ...).
  - View list of students (name, ID, status, ...).
  - View detailed profile of each student.
- **Preconditions:**
  - Teacher is logged in.
- **Postconditions:**
  - Class information and student list are displayed completely and accurately.
- **Normal Flow:**
  1. Teacher accesses the "Class" section.
  2. System displays class overview information.
  3. Teacher views the list of students in the class.
  4. Teacher selects a student to view detailed profile (if needed).
- **Alternative Flows:**
  - No students: System shows message "No students in class".
  - Data loading error: System shows error and keeps the current state.

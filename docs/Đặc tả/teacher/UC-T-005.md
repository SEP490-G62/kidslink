# UC-T-005: Xem thời khóa biểu/lịch dạy

## Tiếng Việt
- **Diễn giải:** Giáo viên chỉ được phép xem thời khóa biểu/lịch dạy của lớp mình.
- **Diễn viên chính:** Giáo viên
- **Diễn viên phụ:** Hệ thống
- **Mô tả:**
  - Xem chi tiết thời khóa biểu/lịch dạy của lớp.
- **Điều kiện tiên quyết:**
  - Giáo viên đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Thời khóa biểu/lịch dạy được hiển thị đầy đủ, chính xác.
- **Luồng chính:**
  1. Giáo viên truy cập mục "Lịch học".
  2. Hệ thống hiển thị chi tiết thời khóa biểu/lịch dạy của lớp.
- **Luồng thay thế:**
  - Không có dữ liệu: Hệ thống hiển thị thông báo "Không có dữ liệu".
  - Lỗi tải dữ liệu: Hệ thống báo lỗi và giữ nguyên trạng thái.

## English
- **Description:** Teacher can only view the class schedule/teaching timetable.
- **Primary Actor:** Teacher
- **Secondary Actor:** System
- **Details:**
  - View detailed class schedule/teaching timetable.
- **Preconditions:**
  - Teacher is logged in.
- **Postconditions:**
  - Schedule/timetable is displayed completely and accurately.
- **Normal Flow:**
  1. Teacher accesses the "Schedule" section.
  2. System displays the detailed class schedule/teaching timetable.
- **Alternative Flows:**
  - No data: System shows message "No data available".
  - Data loading error: System shows error and keeps the current state.

# UC-T-004: Ghi nhận và gửi báo cáo ngày cho học sinh

## Tiếng Việt
- **Diễn giải:** Giáo viên ghi nhận, tổng hợp và gửi báo cáo ngày về hoạt động, sức khỏe, nhận xét cho từng học sinh.
- **Diễn viên chính:** Giáo viên
- **Diễn viên phụ:** Hệ thống
- **Mô tả:**
  - Chọn học sinh cần báo cáo.
  - Nhập thông tin về hoạt động, sức khỏe, nhận xét trong ngày.
  - Gửi báo cáo cho phụ huynh.
- **Điều kiện tiên quyết:**
  - Giáo viên đã đăng nhập hệ thống.
- **Kết quả sau cùng:**
  - Báo cáo ngày được gửi thành công đến phụ huynh.
- **Luồng chính:**
  1. Giáo viên truy cập mục "Báo cáo".
  2. Hệ thống hiển thị danh sách học sinh.
  3. Giáo viên chọn học sinh và nhập thông tin báo cáo ngày.
  4. Giáo viên gửi báo cáo.
  5. Hệ thống xác nhận gửi thành công.
- **Luồng thay thế:**
  - Lỗi gửi báo cáo: Hệ thống báo lỗi và giữ nguyên trạng thái.

## English
- **Description:** Teacher records, summarizes, and sends daily reports about activities, health, and comments for each student.
- **Primary Actor:** Teacher
- **Secondary Actor:** System
- **Details:**
  - Select student to report.
  - Enter information about activities, health, comments for the day.
  - Send report to parent.
- **Preconditions:**
  - Teacher is logged in.
- **Postconditions:**
  - Daily report is successfully sent to parent.
- **Normal Flow:**
  1. Teacher accesses the "Report" section.
  2. System displays the list of students.
  3. Teacher selects student and enters daily report information.
  4. Teacher sends the report.
  5. System confirms successful sending.
- **Alternative Flows:**
  - Error sending report: System shows error and keeps the current state.
